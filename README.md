## 📌 Project Overview

This project was developed as part of the **SmartBridge ServiceNow System Administration Internship 2026**.

The application implements a secure **Role-Based Access Control (RBAC)** system using the **ServiceNow Developer Instance**. It enables administrators to manage users, groups, and roles while automating project workflows through **Flow Designer** and enforcing security using **Access Control Lists (ACLs)**.

The solution improves governance, security, accountability, and operational efficiency by ensuring that every user has appropriate access based on their assigned role.
